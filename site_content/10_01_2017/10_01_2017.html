Using JQuery

Before we begin, you want to make sure have the right tools for the job. You will need to import the JQuery libraries into your project. You may do it by including this snippet under the head tag of your html file.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>

This will import JQuery's libraries from their online storage where the api is stored. It is called a CDN(Content Delivery Network).


Importing a File

Now that we have our JQuery CDN in there, let's look at how we can import a document in our script.
The command that will read from a file into our script is this:
jQuery.get(url, function(data){ });

We first call our JQuery command JQuery.get to, you guessed it, get something for us. We can see that there are two arguments, first the url, and then... another function!? What kind of wizardry is this? Can this only be done under a full moon??? No, not this time young mage; Javascript let's us put functions within functions, it's kind of cool, but not unique to JavaScript. You should look up "lambda expressions", for a start, if you'd like to learn more about this.

Our first argument, url, is pretty much just that, it's a reference to an html or txt file where the information you want to access is stored. In my whole code, I simply made a variable called url and assigned it a value, which was the path to my text file, you can see it in the whole code above.

Now, for the second argument, we simply have another function, which in itself has the argument data. This argument will store all the data from the document we imported, and that's basically that. Within this inner function, you can now use the data from the file however you wanted to. To make things easier for yourself, you might want to use the split() function.




Using split()

If you haven't inferred by the functions name, in short, split() sections your data into sections. It does this by splitting every time it sees the symbol that was passed as an argument. For example, if we use the function data.split(" ") our data will be split on every space, meaning we get individual words. This is then stored as an array.

Take the instance where I used this function:
var lines = data.split("\n");
What lines is now set to is an array of data sectioned off every new line, so lines[0] will return the first line found in the data grabbed from our text file, lines[1] will return the second line, and so on. That should now allow you to work with the documents you imported into your script, you're all set!
